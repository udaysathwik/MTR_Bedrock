MTR Bedrock

Minecraft Transit Railway for Minecraft Bedrock Edition.

Status

🚧 In Development

About

MTR Bedrock aims to bring the Minecraft Transit Railway experience to Minecraft Bedrock Edition.

The project focuses on recreating core railway systems such as stations, routes, depot management, arrival boards, platform screen doors, and the railway dashboard.

Planned Features

- Depot Management
- Stations
- Routes
- Railway Dashboard
- Arrival Boards
- Platform Screen Doors
- Destination Displays
- Next Station Announcements
